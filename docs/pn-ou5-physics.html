2. ACF versus ADF and omega and cool mass fraction

  • The ACF and the ADF are related by ACF / ADF = (1 - ω) / ω
    • Gomez-Llanos:2020a, Garcia-Rojas:2022a
  • For Ou 5 we have ω ∼ 0.5 so this correction is unity
  • But for other nebulae, ω tends to be small so the correction is larger
  • Garcia-Rojas:2022a give ACF and ADF values, but do not give ω directly
    • So we will have to invert the equations to find it
    • ω = 1 / [1 + (ACF / ADF)]
  • Conclusion
    • The lower ADF nebulae (ADF = 9-10) have small ω < 0.1
    • Hf 2-2 has ADF = 18 and has a larger ω

2.1. Cool mass fraction

  • Castañeda-Carlos:2025a prefer to talk of the mass fraction in the cool component, so let's deal with that
  • They find mass fraction μ = 0.18, we want to find emission fraction ω
  • Assume two components, cool and warm: k = {c, w}
  • Recomb line emission from component k: Lk = VEMk hν αeff(Tk) = Vk nk2 hν αeff(Tk)
    • Assume αeff(Tk) = α' Tk-β
  • Mass of component k: Mk = m nk Vk
  • Mass fraction: μ = Mc / (Mc + Mw)
  • Emission fraction: ω = Lc / (Lc + Lw)
  • Volume fraction: ξ = Vc / (Vc + Vw)
    • Vc = ξ V
    • Vw = (1 - ξ) V
  • Mc = m nc Vc = ξ m nc V
  • Mw = m nw Vw = (1 - ξ) m nw V
  • μ = ξ m nc V / [ ξ m nc V + (1 - ξ) m nw V]
    • μ = ξ (nc/nw) / [ ξ (nc/nw) + 1 - ξ]
      • Define δ = nc/nw
    • μ = δ / [δ + ξ-1 - 1]
    • μ = ξ δ / [1 + ξ (δ - 1)]
  • Inverse relation
    • ξ = 1 / [1 + δ (μ-1 - 1)] = μ / [(1 - δ) μ + δ]
    • ξ = μ / [μ + δ (1 - μ)]
  • Lc = Vc nc2 hν α' Tc-β = Mc nc Tc-β (hν α' / m) = μ nc Tc-β (hν α' M / m)
  • Lw = (1 - μ) nw Tw-β (hν α' M / m)
  • ω = μ nc Tc-β / [μ nc Tc-β + (1 - μ) nw Tw-β]
    • Define t = Tc/Tw
    • ω = μ δ t-β / [μ δ t-β + 1 - μ]
    • ω = μ δ t-β / [1 + μ (δ t-β - 1)]
  • Final equation relating mass fraction and emission fraction:
    • ω = A μ /[1 + (A-1) μ]
    • Where A = (nc/nw) (Tc/Tw)-β
  • Put in some numbers for Hf 2-2
    • nw = 500 - 2000 pcc; nc = 3000 - 5000 pcc
      • δ = vmeane([3000, 5000]) / vmeane([500, 2000]) = 3.2 +/- 2.08
    • Tw = 8000 - 16000 K; Tc = 540 K
      • t = 540 / vmeane([8000, 16000]) = 0.045 +/- 0.015
    • β = 0.75 (roughly)
    • A = δ t-β = (3.2 +/- 2.08) (0.045 +/- 0.015)**-0.75 = 33 +/- 23
  • So ω = 33 0.18 / (1 + 32 0.18) = 0.88
    • Or with mean - sigma: (33 - 23) 0.18 / (1 + (32 - 23) 0.18) = 0.69
    • Or with mean + sigma: (33 + 23) 0.18 / (1 + (32 + 23) 0.18) = 0.92
  • Conclusion: ω > 0.7
  • We can also work out the volume fraction: ξ = 0.18 / (0.18 + 33 (1 - 0.18)) = 0.0066
    • Or with mean - sigma: 0.18 / (0.18 + (33 - 23) (1 - 0.18)) = 0.021

6.7. Normalization to the H line luminosity

  • From the extinction-corrected H alpha flux of 3.8e-12 (1 +/- 0.23) erg/s/cm2, then we can calculate the line luminosity to constrain the models:
    • L = (4 pi D**2) F = (4 pi 4000**2 3.085677582e18**2) 3.8e-12 = 7.3e+33 erg/s
    • log L = log10(7.3e+33 (1 +/- 0.23) ) = 33.86 +/- 0.10
  • Although, we need to think about what fraction of the flux comes from the inner shell
    • Down in Summary table of the components we find 74%: log10(0.74) = -0.1308
  • But we also want to divide by the area covering fraction, which is around 0.8, so these will cancel out
    • conclusion best leave it as it was
  • But again [2025-09-09 Tue 07:14] The He II / H I is coming out too low in these models, so we will do the 0.74 factor and also take the -0.1 dex 1 sigma error
    • Use prefix -llha for these:
      • low L(Ha) with target of log10(L(Ha)) = 33.86 - 0.1 - 0.1308 = 33.6292
    • Assuming n ~ sqrt(L) then dex offset is 0.5 (log10(0.74) - 0.1) = -0.1154
    • Or factor of 10**-0.1154 = 0.7667

For us we have λ = 4686 Å => ν = c / λ = 6.39761967563e14 Hz

=> log ν fν = log ν + log fν = -0.4 ABν - 19.44 + 14.8060184187 = -0.4 ABg - 4.6340

c = -log10(F/F0) = -log10(10-0.4A) = 0.4 A => A = 2.5 c

9.1. Graphical Zanstra method from Harman:1966a

https://ui.adsabs.harvard.edu/abs/1966MNRAS.132...15H

The idea is to find the stellar temperature for which the luminosity parameter \[ \Lambda = (L / L_\odot) (D / \mathrm{kpc})^{-2} \] has the same value from two different observations:

  • stellar visual magnitude (for which Λ increases as T increases since fraction of luminosity emitted in visual band is falling)
  • He II recombination line flux (for which Λ decreases as T increases since fraction of luminosity with hν > 4 Rydbergs is increasing)

So if L = 100 Lsun at 4 kpc, we have Λ = 6.25

Everything is calculated in terms of the stellar temperature T*, with \(t = T_* / 10^4 \mathrm{K}\)

They also use a covering factor ξ = Ω / 4 π

20. He II emission

  • We have lots of observations of the He II 6560 line
  • We could compare this with the Ha emission to get the ratio of ionizing photons in the two continua
  • But the trouble with that is that it is not clear if the H+ continuum is optically thick.
    • The [N II] emission is very weak, suggesting it might be thin
  • Alternatively, we could just find a He++ Zanstra temperature
    • How does that work exactly?
    • It requires knowledge of the continuum flux from the star at some reference wavelength, which we cannot do easily in this case because it is a binary
    • We would have to extract it from the light curve somehow, and this will be difficult since most of the flux comes from the cool component I think
