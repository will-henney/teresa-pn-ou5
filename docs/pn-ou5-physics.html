Cool mass fraction

  • Casta√±eda-Carlos:2025a prefer to talk of the mass fraction in the cool component, so let's deal with that
  • They find mass fraction μ = 0.18, we want to find emission fraction ω
  • Assume two components, cool and warm: k = {c, w}
  • Recomb line emission from component k: Lk = VEMk hν αeff(Tk) = Vk nk2 hν αeff(Tk)
    • Assume αeff(Tk) = α' Tk-β
  • Mass of component k: Mk = m nk Vk
  • Mass fraction: μ = Mc / (Mc + Mw)
  • Emission fraction: ω = Lc / (Lc + Lw)
  • Volume fraction: ξ = Vc / (Vc + Vw)
    • Vc = ξ V
    • Vw = (1 - ξ) V
  • Mc = m nc Vc = ξ m nc V
  • Mw = m nw Vw = (1 - ξ) m nw V
  • μ = ξ m nc V / [ ξ m nc V + (1 - ξ) m nw V]
    • μ = ξ (nc/nw) / [ ξ (nc/nw) + 1 - ξ]
      • Define δ = nc/nw
    • μ = δ / [δ + ξ-1 - 1]
    • μ = ξ δ / [1 + ξ (δ - 1)]
  • Inverse relation
    • ξ = 1 / [1 + δ (μ-1 - 1)] = μ / [(1 - δ) μ + δ]
    • ξ = μ / [μ + δ (1 - μ)]
  • Lc = Vc nc2 hν α' Tc-β = Mc nc Tc-β (hν α' / m) = μ nc Tc-β (hν α' M / m)
  • Lw = (1 - μ) nw Tw-β (hν α' M / m)
  • ω = μ nc Tc-β / [μ nc Tc-β + (1 - μ) nw Tw-β]
    • Define t = Tc/Tw
    • ω = μ δ t-β / [μ δ t-β + 1 - μ]
    • ω = μ δ t-β / [1 + μ (δ t-β - 1)]
  • Final equation relating mass fraction and emission fraction:
    • ω = A μ /[1 + (A-1) μ]
    • Where A = (nc/nw) (Tc/Tw)-β
  • Put in some numbers for Hf 2-2
    • nw = 500 - 2000 pcc; nc = 3000 - 5000 pcc
      • δ = vmeane([3000, 5000]) / vmeane([500, 2000]) = 3.2 +/- 2.08
    • Tw = 8000 - 16000 K; Tc = 540 K
      • t = 540 / vmeane([8000, 16000]) = 0.045 +/- 0.015
    • β = 0.75 (roughly)
    • A = δ t-β = (3.2 +/- 2.08) (0.045 +/- 0.015)**-0.75 = 33 +/- 23
  • So ω = 33 0.18 / (1 + 32 0.18) = 0.88
    • Or with mean - sigma: (33 - 23) 0.18 / (1 + (32 - 23) 0.18) = 0.69
    • Or with mean + sigma: (33 + 23) 0.18 / (1 + (32 + 23) 0.18) = 0.92
  • Conclusion: ω > 0.7
